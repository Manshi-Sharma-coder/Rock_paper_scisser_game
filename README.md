## 📌 Project Description
This is a simple and interactive **Rock, Paper, Scissors game** built using Python.  
The player competes against the computer, which makes random choices.

---

## 🎮 How to Play
- Run the Python program
- Enter your choice: `rock`, `paper`, or `scissor`
- Computer randomly selects its move
- The result is displayed instantly

---

## ⚙️ Features
- 👤 User vs Computer gameplay  
- 🎲 Random computer choice using `random` module  
- 🔤 Case-insensitive input support  
- 🎨 Colored terminal output  
- ⚡ Instant result display  

---

## 🧠 Game Rules

- Rock vs Scissor → Rock wins  
- Rock vs Paper → Paper wins  
- Paper vs Scissor → Scissor wins  
- Same choices → Tie  

---

## 🧾 Workflow
1. Import required libraries  
2. Take user input  
3. Generate computer choice randomly  
4. Compare both choices  
5. Display result
