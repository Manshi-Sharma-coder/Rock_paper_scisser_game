# ✊✋✌ Rock Paper Scissors Game (Python)

## 📌 Project Description
This is a simple command-line **Rock, Paper, Scissors game** built using Python.  
The user plays against the computer, which randomly selects its move.

---

## 🎮 How to Play
- Run the program
- Enter your choice: `rock`, `paper`, or `scissor`
- Computer will randomly choose one option
- Result will be displayed (Win / Lose / Tie)

---

## ⚙️ Features
- User vs Computer game
- Random computer choice using Python `random` module
- Case-insensitive input handling
- Colored terminal output (ANSI colors)
- Instant result display

---

## 🧠 Game Rules

- Rock vs Scissor → Rock wins
- Rock vs Paper → Paper wins
- Paper vs Rock → Paper wins
- Paper vs Scissor → Scissor wins
- Scissor vs Rock → Rock wins
- Scissor vs Paper → Scissor wins
- Same choices → Tie

---

## 🧾 Workflow
1. Import libraries (`random`)
2. Take user input
3. Generate computer choice randomly
4. Compare both choices
5. Display result
